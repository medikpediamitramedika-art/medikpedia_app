📋 Checklist Sebelum Submit:

  • ✅ Form memiliki enctype="multipart/form-data"
  • ✅ Form memiliki @csrf dan @method('PUT')
  • ✅ Input file memiliki name="gambar"
  • ✅ Checkbox hapus memiliki name="delete_gambar" value="1"
  • ✅ Route update sudah benar
  • ✅ Controller sudah handle $request->hasFile('gambar')
  • ✅ Controller sudah handle $request->input('delete_gambar')

🔍 Cara Test:

  1. Ganti action="/admin/medicines/1" dengan route yang benar
  2. Ganti YOUR_CSRF_TOKEN_HERE dengan token CSRF yang valid
  3. Pilih foto untuk diupload
  4. Klik "Simpan Perubahan"
  5. Cek apakah foto berhasil diupload ke storage/app/public/medicines/

🐛 Jika Gagal:

  1. Buka Developer Tools (F12)
  2. Lihat tab Network
  3. Submit form
  4. Klik request yang muncul
  5. Lihat Response dan Headers
  6. Screenshot error dan kirim ke developer
