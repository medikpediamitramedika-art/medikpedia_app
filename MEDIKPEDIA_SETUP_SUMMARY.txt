🚀 CARA MENJALANKAN WEBSITE
================================

1. Buka Terminal/Command Prompt
2. Navigate ke folder project:
   cd c:\Users\Ali Attaziri\medicpedia_app

3. Install dependencies (jika belum):
   composer install
   npm install

4. Jalankan development server:
   php artisan serve

5. Buka browser dan akses:
   http://localhost:8000

================================
🔧 CUSTOMIZATION TIPS
================================

1. Mengubah Nama Perusahaan:
   - Ubah "Medikpedia" di navbar (resources/views/layouts/app.blade.php)
   - Update APP_NAME di config/app.php

2. Mengubah Warna Tema:
   - Edit variabel CSS di resources/views/layouts/app.blade.php
   - Ubah hex color codes di :root {}

3. Mengubah Informasi Kontak:
   - Edit footer di resources/views/layouts/app.blade.php
   - Update halaman contact.blade.php

4. Menambah Produk:
   - Duplicate card produk di resources/views/pages/products.blade.php
   - Update gambar, nama, harga, rating

5. Mengubah Harga/Diskon:
   - Edit harga di masing-masing card produk
   - Update package pricing di halaman layanan

================================
🆘 TROUBLESHOOTING
================================

Error 404 - Route not found?
→ Pastikan Anda menjalankan 'php artisan serve'

Styling tidak muncul?
→ Pastikan Bootstrap CDN link aktif di app.blade.php

Database error?
→ Setup .env file dengan database credentials yang benar

Port 8000 sudah terpakai?
→ Gunakan port lain: php artisan serve --port=8001
