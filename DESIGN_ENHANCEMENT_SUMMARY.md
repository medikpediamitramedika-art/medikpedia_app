| Color | Hex Code | Usage |

| Primary Blue | #1E88E5 | Navbar gradient, topbar, form focus, links |
| Secondary Blue | #1565C0 | Gradient accent, hover states |
| Dark Blue | #0D47A1 | Sidebar gradient, footer gradient |

| Light Gray | #f3f4f6 | Page background |

### **Gradient Combinations:**
1. **135° Angle (Navbar, Login/Register, Buttons)**:
   - Start: #1E88E5 (Blue)
   - End: #1565C0 (Dark Blue)
   - Effect: Modern, professional, directional

2. **180° Angle (Sidebar, Footer)**:
   - Sidebar: #0D47A1 → #111827 (Dark Blue to Black)
   - Footer: #1f2937 → #0a0e2e (Dark to Very Dark Blue)
   - Effect: Depth, visual hierarchy

3. **90° Angle (Topbar)**:
   - White → #F5F7FA (Light Gray)
   - Effect: Subtle, professional, minimal
